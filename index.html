Ethan Manilow is a PhD student studying Computer Science working under Bryan Pardo in the Interactive Audio Lab at Northwestern University. He is a musician, coder, and fun person. He refuses to decide whether he loves cats or dogs more.

  • WUT 2017 - Present
    The Web Unmixing Toolbox. An interactive, web-based audio source separation tool for end users. [github] [demo]
  • LabVIEW Hack Computer Simulation 2014
    A functional computer simulation built using only primitive NAND gates entirely LabVIEW. Implemented all primitive and compound logic gates, ALU, registers, clock, and RAM. [demo]
  • SampSyn 2012
    A Mac OSX real-time, granular music synthesizer that creates output based on MIDI input and a user specified audio file. Presented at AES Conference, San Francisco 2012. [github] [demo]
